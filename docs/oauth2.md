# Spring Security

oAuth2是一种协议，提供认证和授权的标准。

实现这种协议的可以是Spring Security，Shrio，甚至自己手动实现。

Spring Security主要有三种角色：

- 第三方应用程序
- 资源所有者
- HTTP服务提供者
  - 认证服务器
  - 资源服务器

客户端授权模式有四种：

- 简单模式
- 授权码模式
- 密码模式
- 客户端模式

访问令牌：

- Access Token
- Refresh Token：刷新token
